# 📊 Finance RAG Chatbot

This is a Retrieval-Augmented Generation (RAG) system that allows users to ask questions from financial PDFs.

---

## 🚀 Features
- Upload PDF
- Ask questions
- Get accurate answers with sources

---

## 🧠 Tech Stack
- Streamlit
- LangChain
- FAISS
- Groq LLM

---

## ▶️ Run Locally

```bash
streamlit run app/app.py
